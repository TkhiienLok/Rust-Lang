# General docs for all of the projects in this repository

Open the offline book locally:

    rustup docs --book

Enter the directory of the project

    cd <project_name>

## Run Project:

    cargo run

or 

    cargo build
    ./<name_of_the_compliled_file>

## Read docs for all dependencies in your project

    cargo doc --open
